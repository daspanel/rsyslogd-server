
# Usage


### Get
```shell
$ docker pull daspanel/rsyslogd-server:latest
```

### Run
```shell
$ docker run --name=my-rsyslogd-server daspanel/rsyslogd-server:latest
```

### Stop
```shell
$ docker stop my-rsyslogd-server
```

### Update image
```shell
$ docker stop my-rsyslogd-server
$ docker pull daspanel/rsyslogd-server:latest
$ docker run --name=my-rsyslogd-server daspanel/rsyslogd-server:latest
```
